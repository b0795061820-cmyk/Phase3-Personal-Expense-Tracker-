# 💰 Personal Expense Tracker

A full-stack **Personal Expense Tracker** application built with:

- **Backend:** FastAPI
- **Database:** SQLite
- **ORM:** SQLAlchemy
- **Authentication:** JWT
- **Frontend:** Flutter
- **State Management:** Bloc / Cubit
- **Architecture:** Clean Architecture

The application allows users to manage their daily expenses, track spending by category, and view analytics through a dashboard.

---

# 🚀 Features

## Authentication
- User Registration
- User Login
- JWT Authentication
- Auto Login using stored token
- Logout functionality

---

## Expense Management
- Add new expense
- View all expenses
- Delete expense
- Categorize expenses

Expense categories include:
- Food
- Transport
- Shopping
- Bills
- Entertainment

---

## Dashboard & Analytics
- Total expenses
- Expenses by category
- Monthly expenses statistics
- Pie Chart visualization
